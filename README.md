하늘빛의 개인 홈페이지, 포트폴리오 [skylightqp.kr](https://skylightqp.kr)

## 시작하기

### 의존성 설치
`npm install`으로 의존성 라이브러리를 설치합니다.

### 빌드
`npm run build`으로 빌드합니다.

### 개발모드
`npm run start`으로 개발 모드를 시작합니다.
